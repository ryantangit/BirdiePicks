# Project
- User functionalities:
	- Log in
	- See user homepage, like OP.gg or U.gg with top champion winrates
	- Access Match history

		- auto completion for text entires to speed up pick time

- Log in / Data analysis / chatbot powered by AI
- Front end: Next.Js
- Backend: Go for API
- Database: MongoDB
- Hosted on AWS
